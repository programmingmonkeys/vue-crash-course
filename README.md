## Vue install

- [Vue.js devtools](https://chrome.google.com/webstore/detail/vuejs-devtools/nhdogjmejiglipccpnnnanhbledajbpd/related?hl=en)
- install vue: `npm i vue`

validate install of vue: `vue --version`
